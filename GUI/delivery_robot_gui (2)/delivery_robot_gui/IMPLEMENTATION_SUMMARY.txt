### Key Implementation Details:

**Closest-First Routing:**
```
Station A (position 1) → Station B (position 2) → Station C (position 3)
Requests automatically sorted by position value
```

**Vision Verification (Simulated):**
- Manager verification: Popup dialog at HOME location
- User verification: Popup dialog at delivery station
- Both trigger automatic lid control

**System State Management:**
- idle: Ready for new requests
- loading: Manager loading items (lid open)
- delivering: Robot en route
- waiting_user: At target, waiting for user verification
- returning_home: All deliveries done, heading back

**Request Lifecycle:**
pending → selected → delivering → completed (or cancelled)
